# Лабораторная работа 0

## Титульный лист

Дисциплина: Системы искусственного интеллекта.

Тема: Знакомство с предобученными нейросетевыми моделями.

Платформа: Hugging Face и библиотека Transformers.

Вариант: 1, определение языка текста.

Номер лабораторной работы: 0.

## Цель работы

Познакомиться с платформой Hugging Face и библиотекой Transformers, найти подходящую предобученную модель для определения языка текста, протестировать её на предложениях на разных языках и оценить качество работы модели.

## Ход выполнения

Для решения задачи выбрана модель `alexneakameni/language_detection`.

Ссылка на карточку модели: `https://huggingface.co/alexneakameni/language_detection`.

Модель выбрана по следующим причинам:

- она предназначена именно для задачи language detection;
- используется через стандартный `transformers.pipeline`;
- поддерживает большое количество языков;
- имеет небольшой размер по сравнению с XLM-RoBERTa-моделью `papluca/xlm-roberta-base-language-detection`;
- в карточке модели указаны метрики качества: accuracy, precision, recall и F1 около `0.973`.

В тестовый набор включены 10 предложений на пяти языках:

- русский;
- английский;
- немецкий;
- французский;
- китайский.

На каждый язык взято по два предложения. Для каждого предложения фиксируется ожидаемая метка языка, предсказанная метка, уверенность модели и признак корректности ответа.

## Код решения

Код решения находится в файле `lab0_language_detection.py`. Этот `.py` файл приложен в папке лабораторной работы и может быть запущен для воспроизведения результатов.

Основные этапы работы скрипта:

1. Загрузка токенизатора и модели через `AutoTokenizer.from_pretrained` и `AutoModelForSequenceClassification.from_pretrained`.
2. Создание пайплайна `pipeline("text-classification")`.
3. Последовательная проверка 10 предложений.
4. Сохранение результатов в `artifacts/language_detection_results.csv`.
5. Сохранение сводки качества и времени работы в `artifacts/language_detection_summary.json`.

## Анализ эффективности модели

Модель подходит для практического использования в базовых задачах определения языка, особенно когда требуется быстро классифицировать короткие текстовые фрагменты. Она работает через стандартный интерфейс Transformers, поэтому её легко встроить в Python-скрипт.

По результатам фактического запуска корректно распознаны все пять проверяемых языков: русский, английский, немецкий, французский и китайский. Ошибок по языкам не обнаружено. Минимальная уверенность среди 10 примеров составила `0.954995`, что также указывает на стабильное качество на выбранном тестовом наборе.

Преимущества выбранной модели:

- простое использование через Hugging Face;
- поддержка большого числа языков;
- небольшой размер модели;
- высокая заявленная точность в карточке модели;
- возможность получать не только метку языка, но и уверенность предсказания.

Ограничения:

- качество может снижаться на очень коротких фразах из одного-двух слов;
- модель может путать близкородственные языки;
- для первого запуска требуется интернет, чтобы скачать модель;
- скорость первого запуска зависит от загрузки модели и кеширования.

## Вывод

В ходе лабораторной работы была выбрана и применена предобученная модель Hugging Face для определения языка текста. Скрипт проверяет 10 предложений на русском, английском, немецком, французском и китайском языках, сохраняет таблицу результатов и формирует итоговую сводку. Работа демонстрирует, что предобученные модели можно использовать для практических задач обработки текста без самостоятельного обучения модели с нуля.
